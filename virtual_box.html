Virtual Box

Qué es Virtual Box ?

Es un programa que permite instalar diferentes sistemas operativos en el computador.

Funcionan de manera virtual, dando la facilidad de probar multiples sistemas operativos en un mismo dispositivo sin arriesgar la integridad del dispositivo y sus sistema operativo principal.

Funciona para los siguientes sistemas operativos:

  • Windows
  • Linux
  • Mac OS X
  • Solaris

Para más información podemos acceder a su página principal:

https://www.virtualbox.org/
